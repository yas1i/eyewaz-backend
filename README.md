# EYEWAZ BACKEND

Flask-Restful-API for our mobile app to help blind people!

1. Create User accounts
2. Upload documents
3. Extract text from documents
4. Extract text from images 
5. Language translation from English to Urdu
6. Create speech Urdu text
7. Folder management
   
We've used the following framework to built this

1. Flask-Restful https://flask-restful.readthedocs.io/en/latest/
2. MongoDB http://docs.mongoengine.org
3. Azure Cognition 
4. Azure Vision
5. Azure File Storage
